# Forest 专注森林助手

### 基本说明

很早之前写的一个 Forest 助手。

有助于在终端内实现 forest 的一些常用操作。

可以在 `Termux` 以及 `VPS`等环境使用。

配合自动任务也可以实现一些 “有趣的功能”。（看你们的想象力了）

### 主要功能

- 获取指定用户 profile

- 手动植树

- 按时间段批量自动植树

- 按时间间隔自动植树

- 领取指定成就

其余的嘛.....你们探索吧。
